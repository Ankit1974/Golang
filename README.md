# Golang Mastery Project

A comprehensive collection of Go (Golang) examples, patterns, and practical exercises. This repository covers everything from basic syntax to advanced concurrency patterns and microservice architecture.

## 🚀 Getting Started

### Prerequisites
- Go 1.18 or higher installed on your machine.
- A basic understanding of programming concepts.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Ankit1974/Golang.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Golang
   ```

## 📂 Project Structure

The project is organized into modular directories, each focusing on a specific Go concept.

### 🧱 Core Basics
- `variable/`: Variable declarations and data types.
- `input/`: Handling user input from the console.
- `conversion/`: Type conversion and casting.
- `pointer/`: Understanding and using pointers.
- `time/`: Working with time, dates, and durations.

### 🔄 Control Flow
- `ifelse/`: Conditional branching logic.
- `switch/`: Multi-way branch statements.
- `loops/`: `for` loops and iteration patterns.

### 📦 Data Structures
- `array/`: Fixed-size arrays.
- `slice/`: Dynamic arrays (slices) and their operations.
- `maps/`: Key-value pairs and hash maps.
- `struct/`: Defining and using custom data structures.

### 🛠️ Functions & Methods
- `function/`: Defining, calling, and return values.
- `method/`: Struct-associated functions.
- `defer/`: Delaying function execution for cleanup.

### 🌐 Web & API Development
- `webRequest/`: Making basic HTTP requests.
- `handlingURL/`: Parsing and manipulating URLs.
- `howToCreateJson/`: Marshaling and Unmarshaling JSON data.
- `howToMakeGetRequest/`: Deep dive into HTTP GET.
- `howToMakePostRequest/`: Deep dive into HTTP POST (JSON/Form).
- `buildingApi/`: A structured approach to building RESTful APIs.

### ⚡ Concurrency & Parallelism
- `concurrency&Goroutines/`: Basics of goroutines.
- `WaitGroup/`: Synchronizing multiple goroutines.
- `mutex/`: Handling race conditions with mutual exclusion.
- `channel/`: Communication between goroutines.
- `ConcurrencyPattern/`: Advanced patterns like Pipeline and Worker Pool.

### 🏗️ Advanced Concepts
- `Microservice/`: Introduction to microservice architecture.
- `middleWare/`: Implementing middleware for web services.
- `errorHandlingPattern/`: Professional error handling strategies.
- `memoryManagment/`: Understanding heap vs stack and GC.
- `testing/`: Writing unit tests and benchmarks.

## 🛠️ Running the Examples

Each directory is typically a standalone package or lesson. To run a specific example:

```bash
cd folder_name
go run main.go
```
